# andersonjunjielu.github.io

Personal academic website for Anderson Junjie Lu (陸俊捷), PhD candidate in Epidemiology and Clinical Research at Stanford.

## Files

- `index.html` — the website
- `images/portrait.jpg` — hero portrait
- `cv/Anderson-Junjie-Lu-CV.pdf` — downloadable CV

## Publishing on GitHub Pages

1. Create a **public** repository on GitHub named exactly `andersonjunjielu.github.io`
2. Upload the contents of this folder (the `index.html`, plus the `images/` and `cv/` folders) to the repo root
3. In the repo → **Settings** → **Pages** → under "Source" select `main` branch, `/ (root)`, and Save
4. Wait ~1 minute — the site goes live at **https://andersonjunjielu.github.io/**

## Updating later

Edit `index.html` (or replace the CV / portrait), then either:
- Drag the updated file into the repo on github.com (easiest), or
- Use `git push` from your terminal

Every push redeploys the site automatically.
